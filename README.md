# Power-Plant-Design-and-Off-design
This repository includes the basic subcritical Organic Rankine cycle (ORC) code as well as the design and off-design simulation codes. For additional details, please refer to our papers at https://doi.org/10.1016/j.ecmx.2026.101556 and https://www.yandy-ager.com/index.php/ager/article/view/1982. The citation is:
